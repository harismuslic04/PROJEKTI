Danas radimo CSS(Cascading Style Sheets) - Kaskadne stilove

Postoji 3 nacina da primenimo stilove na neki sadrzaj unutar naseg web sajta.

  1. Inline CSS. CSS koji se unosi kao atribut nekog html taga.
  2. Stilovi definisani unutar style taga, koji se nalazi u head tagu.
  3. Eksterni CSS. Fajl koji je formata .css i njega povezujemo sa html fajlom naseg web sajta.
